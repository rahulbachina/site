ABOUT PHOTOGRAM

    • Photogram is a source code package using the Ionic Framework, TypeScript and Sass to develop its front end, with the Open Source Parse Server to be its backend with NodeJS, Express and MongoDB.

    • Inspired by Instagram, and has many amazing features, such as sharing photos, friends, facebook login, real-time chat, and more.

    • It is suitable for developers and startups who have the minimum knowledge of Javascript, Ionic Framework and NodeJS.
